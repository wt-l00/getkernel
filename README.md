# getkernel

## what

A tool to search and download kernels for ubuntu


## how to install

``` 
go install github.com/wt-l00/getkernel@latest
```

## list

```
getkernel list
```

## fetch

```
getkernel fetch vx.x.x #e.g. v5.8.7
```
